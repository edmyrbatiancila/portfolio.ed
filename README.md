## ğŸš€ Getting Started

### Prerequisites

### Installation

1. Clone the repository:
```bash
git clone https://github.com/edmyrbatiancila/portfolio.ed.git
cd portfolio.ed
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
